Rotina dinâmica de apredizado

Ao entrar na Trybe, tive uma grata surpresa do quão dinâmica e prática são as aulas e conteúdos abordados, tudo é organizado para nós manter engajados. Ainda mais, nos somos ensinados tanto habilidades técnicas quanto competências de soft skills essenciais para nossa atuação profissional.

O dia a dia na Trybe nós imerge diretamente na rotina de trabalho do desenvolvedor. Essa experiência é alcançada de diversas forma: a comunicação entre profissionais e alunos é pelo Slack; o uso contínuo de Git e GitHub nos nossos exercícios e projetos; momentos de one-a-one com facilitadores; e muito mais.

Referências para Desenvolvedores

Sites Descrição
W3 Schools Um dos maiores e mais tradicionais sites de conteúdo técnico sobre desenvolvimento web. Possui desde tutoriais com exercícios até explicações sobre cada feature ou elemento de HTML, CSS e Javascript.
MDN Web Docs Um ótimo site de ensino e blog de notícias sobre desenvolvimento web. Possui cursos gratuitos completos de HTML e CSS, e também um glossário sobre os elementos HTML e propriedades CSS. Criado e mantido pela Mozilla.
HTML Reference.io Um dicionário muito prático de referência para elementos HTML, super rápido e simplificado, permite pesquisar eficientemente se existe o elemento HTML que você procura.
Gitignore.io Ferramenta para criar arquivos gitignore que atendam as suas necessidades pessoais. Ele tem uma vasta lista de aplicativos e tipos de arquivos que podem ser ignorados. Só escolher seus alvos e ele te dá um gitignore prontinho para uso.
PicResize Edite, corte e redimensione suas figuras do jeito que desejar. Gratuito para uso limitado. Sempre bom ter um editor de imagens a disposição.
Scheme Color Compilador e buscador de esquemas de cores, seu recurso mais interressante é oferecer esquemas baseados em temas específicos, como: casamentos, estações, decorações, lugares e até imagens de referência.
Coolors.co Criador prático e rápido de paletas de cores. Possui um algoritmo que recebe algumas cores iniciais e te sugere outras cores que juntas criem uma paleta. Também pode gerar automaticamente paletas.

Um diferencial na educação profissional

Dentre as experiências vividas na Trybe, algumas das mais enriquecedoras são as dinâmicas de soft skills. Considero-as um grande diferencial porque não existe essa abordagem na educação padrão, assim, acho muito enriquecer os soft skills ensinados e como somos colocados para praticá-los em equipe.
